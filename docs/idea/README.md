# Idea Notes

This directory contains **internal idea sketches** and exploratory notes.

These documents are intentionally speculative. They are useful for preserving potential directions, but they are **not commitments** and they are **not implementation source of truth**.

When deciding what Fission currently is or where it is officially heading, prefer:

1. [`README.md`](../../README.md)
2. [`docs/architecture/ARCHITECTURE.md`](../architecture/ARCHITECTURE.md)
3. [`docs/ROADMAP.md`](../ROADMAP.md)
4. [`docs/changelog/CHANGELOG.md`](../changelog/CHANGELOG.md)

Typical contents here:

- product ideas
- engine experiments
- integration sketches
- UI or workflow concepts
- longer-term research directions

Use this folder as a place for exploration, not as a public contract.
