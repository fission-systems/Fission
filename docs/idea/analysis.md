## 1. 테인트(Taint) 분석

- **목표**: 취약점 추적 자동화.
- **세부 내용**: 소스(Source)에서 싱크(Sink)까지의 데이터 오염 전파 추적.
- **날짜**: 2026-01-10

## 2. 심볼릭 실행 통합

- **목표**: 난독화 해제 및 정밀 분석.
- **세부 내용**: 불투명 조건자 제거 및 상수 폴딩을 위한 로컬 심볼릭 실행 엔진 통합.
- **날짜**: 2026-01-10

## 3. 바이너리 디핑 (Binary Diffing)

- **목표**: 패치 분석 및 코드 변경 확인.
- **세부 내용**: 두 바이너리 간의 제어 흐름 및 기본 블록 비교 시각화 (Diaphora 스타일).
- **날짜**: 2026-01-10

## 4. 협업 리버싱

- **목표**: 팀 단위 분석 효율성 증대.
- **세부 내용**: 실시간으로 여러 사용자가 동일한 프로젝트를 분석하고 주석/이름을 공유하는 세션 지원. (장기 목표)
- **날짜**: 2026-01-10
