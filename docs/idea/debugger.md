# Debugger Integration Ideas (2026-01-10)

> ⚠️ **Status: Stale / Historical Idea Note**
> This note captures exploratory debugger ideas from 2026-01-10.
> It is not an active roadmap document.

## 1. 리버스 디버깅 (RR/TTD)

- **목표**: 디버깅 효율성의 획기적 개선.
- **세부 내용**: 타임 트래블 디버깅(Time Travel Debugging) 백엔드와 연동하여 실행 흐름을 되감기 가능한 기능 구현.
- **날짜**: 2026-01-10

## 2. 원격 디버깅

- **목표**: 임베디드 및 원격 시스템 분석.
- **세부 내용**: GDB/LLDB 직렬 프로토콜 등을 명시적으로 지원하여 원격 타겟 디버깅.
- **날짜**: 2026-01-10

## 3. 커널 디버깅

- **목표**: OS 레벨 분석 지원.
- **세부 내용**: WinDbg/KB 프로토콜을 활용하여 커널 모듈 및 드라이버 분석/디버깅 지원.
- **날짜**: 2026-01-10
