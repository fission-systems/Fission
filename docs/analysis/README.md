# Analysis Notes

This directory contains **internal analysis notes** about decompiler behavior, experiments, failure modes, and validation work.

These files are useful for understanding historical investigations, but they are **not source of truth** for the current architecture or product direction.

When in doubt, prefer:

1. [`README.md`](../../README.md)
2. [`docs/architecture/ARCHITECTURE.md`](../architecture/ARCHITECTURE.md)
3. [`docs/ROADMAP.md`](../ROADMAP.md)
4. [`docs/changelog/CHANGELOG.md`](../changelog/CHANGELOG.md)

Typical contents here:

- root-cause analysis for specific regressions
- quality/readability experiments
- FID/type propagation investigations
- postprocess and canonicalization notes
- historical validation logs

Treat this folder as **reference material** and **working notes**, not as a stable public contract.
